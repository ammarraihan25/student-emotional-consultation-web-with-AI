====================================================================
PANDUAN SETUP LOKAL PROYEK WEB KONSELING EMOSIONAL AI - KELOMPOK 3
====================================================================

Proyek ini menggunakan struktur Monorepo (Laravel 13.7 untuk Web dan Python untuk AI), ada beberapa langkah yang harus dilakukan setelah melakukan 'git clone' agar aplikasi bisa berjalan di laptop masing-masing.

Pastikan laptop kalian sudah terinstal:
- Laragon
- Composer
- PHP versi 8.4.5 (biar sama)
- apache 2.4.66 (bawaan laragon, cek dulu)
- Node.js versi 24 keatas (untuk NPM & TailwindCSS)
- Python (disarankan versi 3.14 biar sama)

--------------------------------------------------------------------
TAHAP 1: CLONE REPOSITORY
--------------------------------------------------------------------
1. Buka terminal/CMD di dalam folder C:\laragon\www
2. Jalankan perintah:
   git clone https://github.com/devannoap31/student-emotional-consultation-web-with-AI.git
3. Masuk ke folder proyek:
   cd student-emotional-consultation-web-with-AI
4. Buka folder ini menggunakan VS Code.

--------------------------------------------------------------------
TAHAP 2: SETUP MESIN AI (PYTHON)
--------------------------------------------------------------------
Karena folder 'venv' tidak ikut di-upload ke GitHub, kalian harus meraciknya sendiri:

1. Buka terminal baru di VS Code, masuk ke folder AI:
   cd ai-konseling
2. Buat Virtual Environment baru:
   python -m venv venv
3. Aktifkan venv tersebut:
   - Windows (CMD/Powershell): venv\Scripts\activate
   - Mac/Linux: source venv/bin/activate
   (Pastikan muncul tulisan '(venv)' di terminal)
4. Install library yang dibutuhkan:
   pip install fastapi uvicorn requests pydantic python-dotenv
5. Buat file baru bernama persis: .env (di dalam folder ai-konseling)
6. Isi file .env tersebut dengan API Key Gemini masing-masing (Bisa buat gratis di Google AI Studio):
   GEMINI_API_KEY=masukkan_api_key_kalian_disini
7. Nyalakan server AI:
   uvicorn main:app --reload --port 8000
   (Biarkan terminal ini tetap menyala)
   lalu jika ingin menutup service ini tekan ctrl+c di cmd, lalu jalankan perintah 'deactivate' untuk menonaktifkan venv.
   jangan lupa setiap ingin menjalankan projek ini, aktifkan dulu venv seperti di langkah 3, baru ketik perintah seperti di langkah 7.

8. CARA MENGETES SERVER AI (SWAGGER UI):
   - Buka browser dan ketik alamat: http://127.0.0.1:8000/docs
   - Klik baris hijau bertuliskan "POST /analyze-emotion".
   - Klik tombol "Try it out" di sebelah kanan.
   - Di dalam kotak "Request body", ubah teks "string" menjadi kalimat curhatan.
   - Klik tombol biru "Execute".
   - Scroll ke bawah, respons dari sistem CBT Gemini akan muncul di bagian "Server response".

--------------------------------------------------------------------
TAHAP 3: SETUP WEB FRONTEND & BACKEND (LARAVEL)
--------------------------------------------------------------------
Karena folder 'vendor' dan 'node_modules' tidak di-upload, ikuti langkah ini:

1. Buka Tab Terminal BARU di VS Code, masuk ke folder web:
   cd web-konseling
2. Install dependensi PHP:
   composer install
3. Install dependensi Node.js (untuk Tailwind CSS):
   npm install
4. Buat file environment Laravel:
   - Copy file .env.example lalu paste dan ubah namanya menjadi .env
5. Generate Application Key Laravel:
   php artisan key:generate
6. Setup Database:
   - Buka HeidiSQL / phpMyAdmin di Laragon.
   - Buat database baru dengan nama: db_konseling
   - Buka file .env (yang ada di folder web-konseling), pastikan baris database seperti ini:
     DB_CONNECTION=mysql
     DB_HOST=127.0.0.1
     DB_PORT=3306
     DB_DATABASE=db_konseling
     DB_USERNAME=root
     DB_PASSWORD=
7. Jalankan migrasi tabel database:
   php artisan migrate
8. Nyalakan mesin compile CSS (Tailwind):
   npm run dev
   (Biarkan terminal ini tetap menyala)
9. Akses web-konseling dengan menjalankan perintah berikut:
   php artisan serve --port=8080
   (Biarkan terminal ini tetap menyala)
   
lalu jika ingin menutup service ini cukup tekan ctrl+c di terminal yang menjalankan 'npm run dev' dan 'php artisan serve --port=8080'.
jangan lupa setiap ingin menjalankan projek ini, ketik perintah seperti di langkah 9.

Jangan lupa selalu 'git pull' sebelum mulai bekerja agar kode tidak bentrok.
